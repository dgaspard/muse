# Error Model v1.0

## Purpose
Defines the canonical error object and error code enumeration for all governance validation and artifact creation in MUSE.

---

## Canonical Error Object
```json
{
  "status": "error",
  "severity": "blocking | warning",
  "code": "INVALID_LINEAGE",
  "artifact_type": "story",
  "artifact_id": "STORY-8c41aa77",
  "message": "Human-readable explanation",
  "action": "What the user should do next"
}
```

- `status`: always "error"
- `severity`: "blocking" or "warning"
- `code`: stable, enumerable error code
- `artifact_type`: epic | feature | story | prompt
- `artifact_id`: stable artifact ID
- `message`: human-readable
- `action`: required for blocking errors

## Error Codes (examples)
- MISSING_EPIC_ID
- INVALID_LINEAGE
- MISSING_GOVERNANCE_REFERENCE
- SCHEMA_VIOLATION
- DUPLICATE_ARTIFACT
